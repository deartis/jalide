Sobre o projeto

JALIDE transforma seu Android em uma estação de desenvolvimento portátil.

Uma IDE mobile moderna, criada em Flutter, para programar, testar, editar arquivos e acessar servidores usando apenas o celular.

Ver repositório Voltar para a landing

Por que o JALIDE existe?

O JALIDE nasceu com uma ideia simples: permitir que qualquer pessoa desenvolvedora consiga continuar produtiva longe do computador. Seja para corrigir algo rápido em produção, estudar no ônibus ou tocar projetos completos pelo Android, a IDE foi pensada para unir praticidade e produtividade.

O foco inicial é desenvolvimento JavaScript e Node.js no Android, com uma experiência confortável para editar código, navegar por arquivos, usar terminal e trabalhar com conexões remotas.

Desenvolvimento real no celular

Editor, terminal, SSH, SFTP e integração com Termux em um fluxo pensado para programação mobile.

O que a IDE oferece

Editor profissional

Syntax highlighting, múltiplas abas, zoom dinâmico e autosave para uma edição mais fluida.

Terminal integrado

Sessões locais e acesso SSH para trabalhar com projetos e servidores diretamente do Android.

Termux Magic

Configuração facilitada de Node.js, Git, OpenSSH e ambiente Linux para desenvolvimento.

SFTP nativo

Edite arquivos remotos como se fossem locais, com perfis SSH salvos de forma prática.

Integração

JALIDE + Termux

Para liberar o potencial máximo da IDE, o projeto usa o Termux como ponte para Node.js, NPM, Git, SSH e automações Linux. O fluxo "Termux Magic" ajuda a preparar o ambiente com menos atrito.

  1. Abra o painel de terminal no JALIDE.
  2. Use o botão de configuração rápida.
  3. Copie o comando para o Termux.
  4. Conecte via SSH e comece a desenvolver.
