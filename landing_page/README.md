# JALIDE Landing Page

Esta é uma landing page simples e moderna para o projeto JALIDE.

## Como visualizar
1. Abra o arquivo `index.html` diretamente em seu navegador.
2. Ou use uma extensão como "Live Server" no VS Code.

## Customização
- **Imagens:** Substitua a div `.screen-placeholder` em `index.html` por uma imagem real `<img src="path/to/screenshot.png">` do app.
- **Cores:** As cores principais podem ser alteradas no topo do arquivo `style.css` na seção `:root`.
- **Links:** Atualize os links de download para apontar para o seu repositório de APKs ou loja.
